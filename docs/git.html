Git Cheat Sheet

Init 

git init

Stage 

// add all file
git add *

// with wildcard
git add dir/*.txt

Unstage/Reset 

git reset filename

//reset all
git reset

Commit 

git commit -m "comment for commit"

Branch 

// create bracnh and switch to its branch
git checkout -b branchName

// long version
git branch branchName
git checkout branchName

Merge 

// Back to master and apply merge your branch into master
git checkout master
git merge hotfix

// delete your branch if necessary
git branch -d branchName

Resolve conflict with mergetool and commit it 

// open 3rd party merge tool
git mergetool

// commit changes
git commit -a -m "message to commit"
